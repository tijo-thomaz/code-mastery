Thanks for wanting to contribute! This repo is built by the community, for the community.

## How to Contribute

### 1. Fork & Clone
```bash
git fork https://github.com/YOUR_USERNAME/code-mastery.git
git clone https://github.com/YOUR_USERNAME/code-mastery.git
cd code-mastery
```

### 2. Create a Branch
```bash
git checkout -b add/my-exercise
```

### 3. Add Your Content

**Adding Exercises:**
- Place in the correct `language/exercises/` folder
- Follow the existing numbering pattern
- Include clear instructions as comments
- Difficulty: mark as `// Difficulty: Easy | Medium | Hard`

**Adding Solutions:**
- Match the exercise filename in `language/solutions/`
- Include comments explaining *why*, not just *what*

**Adding Interview Questions:**
- Place in `language/interview-questions/` or `failed-interviews/`
- Include: the question, your attempt, the correct answer, and what you learned

**Adding Speed-Typing Snippets:**
- Place in `language/speed-typing/`
- Keep snippets focused (5-15 lines each)
- Group by pattern or concept

### 4. Submit a PR
```bash
git add .
git commit -m "add: JS closure exercises"
git push origin add/my-exercise
```

Then open a Pull Request with a clear description.

## Guidelines

- Keep code clean and well-commented
- No plagiarism ‚Äî write original exercises or credit sources
- Be respectful in all interactions
- Test your solutions before submitting
